# Study Optimizer

TOEICや大学の勉強を、予定・学習ログ・忘却曲線に合わせて最適化する学習アプリです。

## 目的

- TOEIC 800点を目指す
- 大学の授業・課題・試験勉強も管理する
- 空き時間に合わせて「今やるべき勉強」を提示する
- 学習ログから苦手分野を分析する
- 復習タイミングを自動で決める

## 最初に作る機能

### 1. 学習タスク管理

- 勉強する内容を登録する
- 締切や試験日を登録する
- 優先度をつける

### 2. 学習ログ

- 何を何分勉強したか記録する
- 正答率や理解度を記録する
- 苦手分野を見える化する

### 3. 復習スケジュール

- 忘却曲線に基づいて復習日を決める
- 未復習の内容を一覧表示する

### 4. カレンダー連携

- 空き時間を見つける
- 空き時間に合わせて学習内容を提案する

## 使いたい技術

- Next.js
- TypeScript
- Supabase
- Google Calendar API
- ChatGPT / Codex
