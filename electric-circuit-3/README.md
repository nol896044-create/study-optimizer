# 電気回路Ⅲ 学習資料

このフォルダは，電気回路Ⅲの試験対策・解説資料・LaTeXソース・図表・データを管理するための作業場所です。

## 目的

- 電気回路Ⅲの内容を基礎から理解する
- 期末試験に向けて，出題範囲を整理する
- LaTeXで解説PDFを作成・修正する
- 図，データ，問題リストをまとめて管理する
- ChatGPT / Codex を使って資料を継続的に改善する

## フォルダ構成

```text
electric-circuit-3/
├─ README.md
├─ tex/
│  └─ main.tex
├─ pdf/
├─ figures/
├─ notes/
│  ├─ exam-info.md
│  ├─ weak-points.md
│  └─ problem-list.md
└─ data/
```

## 管理する内容

- `tex/`：LaTeXソース
- `pdf/`：出力したPDF
- `figures/`：回路図・グラフ・説明図
- `notes/`：試験情報，苦手分野，問題リスト
- `data/`：gnuplotや表計算で使うデータ
